Как уже описывалось ранее, медленное многообразие $M_0$ плоской эллиптической ограниченной задачи трех тел в окрестности резонанса 3:1 состоит из двух листов $M_{0,+}$, $M_{0,-}$ и двух перемычек $M_{0,b}^{\pm}$, причем лист $M_{0,+}$ состоит из устойчивых положений равновесия быстрой системы, а лист $M_{0,-}$ — из неустойчивых.

Для дальнейшего анализа существенно, что неустойчивые положения равновесия на $M_{0,-}$ обладают двумерными сепаратрисами, параметризуемыми решениями вида (\ref{fast_sep}).

Также отметим, что для любой траектории медленной системы (\ref{slowreal}) на $M_{0,-}$ вида $\left( x(t), y(t), 0, \lambda_{-} \left( x(t),y(t) \right), \Lambda = 0 \right)$ окрестность этой траектории на плоскости $xy$ образует инвариантное гиперболическое множество.

Для построения функции Мельникова в исследуемой системе выполним приведение уравнений (\ref{fullt}) к виду (\ref{wiggs}) с помощью следующей замены переменных:
$$
X = \begin{pmatrix}
\Lambda\\
z
\end{pmatrix}, \quad
I = \begin{pmatrix}
x\\
y
\end{pmatrix},
$$
где $z = \lambda - \lambda_{-}(x,y)$. 

В новых переменных многообразие $M_{0,-}$ принимает вид:
$$\tilde M_{0,-} = \left\{ (\Lambda, z, x, y): \Lambda = 0,\ z = 0 \right\},$$
а сепаратрисные решения записываются как:
$$z_{sep}(t,t_0) = \pm 2 \arctan{\sh \left( \alpha \beta (t-t_0) \right)}, \quad \Lambda_{sep}(t,t_0) = \frac{2\beta}{\ch \left(\alpha \beta (t-t_0) \right)}.$$

Выражения для операторов и функций системы в новых переменных:
$$
D_X = \begin{pmatrix}
\frac{\partial}{\partial \Lambda}\\
\frac{\partial}{\partial z}
\end{pmatrix}, \quad
D_I = \begin{pmatrix}
\frac{\partial}{\partial x}\\
\frac{\partial}{\partial y}
\end{pmatrix},
$$
$$H_1 = \alpha \frac{\Lambda^2}{2} + \beta(x,y)(1 - \cos z), \quad \beta(x,y) = \sqrt{U(x,y)^2 + V(x,y)^2},$$
$$g^I = \begin{pmatrix}
-2Fy + \frac{\partial \beta}{\partial y} \cos z + \beta(x,y) \frac{\partial}{\partial y} \arctan\left(\frac{V}{U}\right) \sin z \\
2Fx + e_JG - \frac{\partial \beta}{\partial x} \cos z - \beta(x,y) \frac{\partial}{\partial x} \arctan\left(\frac{V}{U}\right) \sin z
\end{pmatrix}, \quad g^X = \frac{-1}{\varepsilon} \begin{pmatrix}
0 \\
\frac{d}{dt} \arctan\left(\frac{V}{U}\right)
\end{pmatrix}.$$

\subsection{Расщепление сепаратрис}

Вычисление функции Мельникова для множества $V = \tilde M_{0,-}$ приводит к следующему выражению:
\begin{equation}
M^I(x,y) = \frac{4}{\alpha \beta^2} \left( 
U \left( (2Fx + e_J G)\frac{\partial U}{\partial y} - 2Fy \frac{\partial U}{\partial x} \right) + 
V \left((2Fx + e_J G)\frac{\partial V}{\partial y} - 2Fy \frac{\partial V}{\partial x} \right)
\right).
\end{equation}

Множество нулей функции Мельникова определяется уравнением:
\begin{equation}
0 = y \left( 
(2x + \tilde{\alpha})^2 + y^2 - (2x + \tilde{\alpha})\frac{4FK}{C(e_JG - F \tilde{\alpha})} - \frac{K}{C}
\right),
\end{equation}
которое описывает прямую $y = 0$ и эллипс (рис. \ref{sep_zero}).

Анализ градиента функции Мельникова показывает, что он обращается в нуль в двух точках данного множества, что свидетельствует о наличии трансверсальных пересечений.

На основании полученных результатов можно сделать следующие выводы:
\begin{enumerate}
\item Для любой орбиты медленной системы на листе $M_{0,-}$ и её достаточно малой окрестности, не содержащей особых точек, при малых $\varepsilon > 0$ существует локально инвариантное гиперболическое множество с трансверсальным пересечением устойчивого и неустойчивого многообразий (гарантируется теоремами Феничеля).

\item  Положение равновесия $\left( x_{0}^{-}, y_{0}^{-}, 0, \lambda_{-}(x_{0}^{-}, y_{0}^{-}) \right)$ является гиперболическим для всех $\varepsilon > 0$, причем при малых возмущениях соответствующие ему устойчивое и неустойчивое многообразия пересекаются трансверсально.
Гиперболичность положения равновесия $\left( x_{0}^{-}, y_{0}^{-}, 0, \lambda_{-}(x_{0}^{-}, y_{0}^{-}) \right)$ сохраняется при возмущении, а трансверсальность пересечения сепаратрис следует из анализа декартова произведения сепаратрис быстрой и медленной систем с учетом пересечения медленной сепаратрисы с множеством нулей функции Мельникова (рис. \ref{sep_zero}).
\end{enumerate}
